Project Overview

The AutonoMe project consisted in the designing, implementing and shipping to production a web application for people with autism.

This application was built with a focus on accessibility and user experience, providing essential features to help users navigate and interact with the platform independently.

Features

  • Help button with Stimulus JS
  • Geographical position and route via import-maps
  • Authentication with Devise
  • Responsive Bootstrap design
